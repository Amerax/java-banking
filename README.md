# java-banking

Hi this is my java banking project written in java of course.
It is quite nice and unique because it has multiple classes and also has a checkign and savings account. 
Also it has a random unique accoutn number function which makes sure each account has a unqiue identifier. 

Also this features the ability to make multiple accounts and create multiple instances so that you can play around and have a similar software experience to a real bank. 

There is both a main menu portal in the terminal which you must go through to get into your account and then once you log in there you can perform the basic actions.
